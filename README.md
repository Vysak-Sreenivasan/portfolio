# Portfolio Website

A modern, responsive portfolio website showcasing AI engineering expertise with a sleek black and white monochrome theme.

## Features

- **Monochrome Design**: Sophisticated black and white aesthetic
- **Particle Animation**: Interactive particle.js background
- **Smooth Animations**: Shimmer, pulse, glow, and 3D transform effects
- **Responsive**: Fully responsive design for all devices
- **Interactive**: Enhanced hover effects and smooth scrolling
- **Stats Counter**: Animated statistics display
- **Contact Form**: Functional mailto integration

## Structure

```
Portfolio/
├── index.html              # Main HTML file
├── css/
│   └── styles.css         # All styling and animations
├── js/
│   └── script.js          # JavaScript functionality
└── assets/
    └── images/            # Image assets (if any)
```

## Technologies Used

- HTML5
- CSS3 (with advanced animations)
- Vanilla JavaScript
- Particles.js
- Font Awesome Icons
- Google Fonts (Inter, JetBrains Mono)

## Local Development

1. Clone or download this repository
2. Open `index.html` in your browser
3. No build process required - pure HTML/CSS/JS

## Deployment

This portfolio can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Simply upload all files maintaining the directory structure.

## Customization

- **Colors**: Edit CSS variables in `css/styles.css`
- **Content**: Update text and links in `index.html`
- **Animations**: Modify keyframes and transitions in `css/styles.css`
- **Particles**: Configure settings in `js/script.js`

## Browser Support

- Chrome (recommended)
- Firefox

- Opera
